# Создание плагина

Чтобы сделать плагин совместимым с tgsnake, ваш плагин должен соответствовать нашему руководству.

- Плагины должны быть функциями.
- Для плагина входа присвойте своей функции префикс `Login`, и она должна возвращать результат из `getMe`.

## Пример

```ts
import { type Snake } from 'tgsnake';
import { Clients } from '@tgsnake/core';
export async function LoginWithA(client: Snake) {
  // you code
  return await Clients.Auth.getMe(client._client);
}
```

Эта процедура может измениться в будущем в зависимости от потребностей и эффективности кода.
