# Available methods

You can directly use the methods here in the `TLObject` or `Snake` derived classes.

```ts {2,5}
const client = new Snake();
client.api; //.<method_name>
// or
bot.on('message.text', (update) => {
  update.api; //.<method_name>
});
```

## Invoking raw api

```ts
import { Raw } from 'tgsnake';
client.api.invoke(new Raw.messages.SendMessage({}));
```

For a list of Raw Api, you can find it in [deno documentation version](https://deno.land/x/tgsnake_core/src/index.ts?s=Raw) and as a supporter, you can read more in the official [telegram documentation.](https://corefork.telegram.org/schema)
