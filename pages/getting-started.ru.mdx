# tgsnake

**tgsnake** — это MTProto библиотека для JavaScript или TypeScript с некоторыми дополнительными функциями и возможностями, которые вы можете не найти в других платформах.

## Начало работы

```bash
npx create-tgsnake-app myapp --ts
```

Команда выше создаст приложение tgsnake, которое находится в папке `myapp`, используя TypeScript.

```bash
create-tgsnake-app location [options]
```

`location` — это место, где будет создан tgsnake. Местоположение должно быть пустой папкой. В противном случае приложение не удастся создать.

| Название опции | Параметр | Описание |
| :-: | :-: | :-: |
| `--ts` or `--typescript` |  | Создайте приложение tgsnake, используя TypeScript. |
| `--yarn` or `--use-yarn` |  | Использует `yarn` для установки необходимых зависимостей. |
| `--npm` or `--use-npm` |  | Использует `npm` для установки необходимых зависимостей. |
| `--pnpm` or `--use-pnpm` |  | Использует `pnpm` для установки необходимых зависимостей. |
| `--rds` or `--use-redis-session` |  | Создает приложение tgsnake, используя `@tgsnake/redis-session` библиотеку вместо встроенной сессии. |
| `--template` | `simple` or `composer` or `git(hub)?` url | Используйте предоставленный шаблон или URL-адрес `git(hub)?`, который будет клонирован в этот каталог. |
