# Invoke

Requesting function using raw api.  
Raw api is a function constructor which is declared in the telegram schema.

## Usage

```ts
import { Raw } from 'tgsnake'; // '@tgsnake/core'

client.api.invoke(query, retries, timeout, sleepTreshold);
```

| name | required | type | description |
| :-: | :-: | :-: | :-- |
| query | Yes | Object | Constructor (object) which will be sent to the Telegram server. |
| retries | No | Integer | If the function fails, how many attempts should it run. |
| timeout | No | Integer | The maximum time limit for running the `query` function. |
| sleepTreshold | No | Integer | Pause sleep time if a flood occurs. |
